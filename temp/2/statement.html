PROBLEM B - Pemilu

Time Limit: 3s

Musim Pemilu sudah tiba, bantulah prof oak untuk menentukan kandidat manakah yang akan memenangkan pemilu ini berdasarkan hasil pemilihan daerah.

Format Masukan

Baris pertama berisi sebuah bilangan T (1 <= T <= 1000), yaitu banyak test case yang diujikan. Setiap test case dimulai dengan sebuah bilangan N (2 <= N <= 5) dan M (0 <= M <= 100) yang menandakan banyaknya kandidat dan banyaknya daerah pemilihan. Sebanyak M baris berikutnya berisi N buah bilangan, v1, v2, ..., vN (0 <= vi <= 1000) banyaknya suara untuk kandidat ke-i.

Format Keluaran

Sebuah baris untuk tiap test case dengan format "Case #x: y" dimana x adalah nomor kasus dan y adalah nomor kandidat pemenang pemilu.

Contoh Masukan

 
2
3 3
159 213 450
512 890 993
215 420 397
2 5
40 66
35 12
102 58
43 15
79 41

Contoh Keluaran

 
Case #1: 3
Case #2: 1
