Max Score

Time Limit: 3s

Description

Delphi really enjoys arithmetic and one day her teacher devised the following problem.

Given n integers a1, a2, …, an and a sum s initialized to 0, Delphi has to perform the following operation for i = 1, 2, …, n in order:

Add ai to s, if the resulting s is even, Delphi earns 1 point and repeatedly divides s by 2 until s becomes odd.

Note that Delphi can earn at most 1 point per operation, regardless of how many divisions she does.

Since these divisions are considered more beneficial for Delphi’s development, her teacher wants to rearrange a so that the number of Delphi’s total points is maximized. Determine the maximum number of points Delphi can gain.

Input Format

The first line contains a single integer n (1 ≤ n ≤ 100), the number of integers.
The second line contains n space-separated integers a1, a2, …, an (1 ≤ ai ≤ 109).

Output Format

Output a single integer, which is the maximum number of points Delphi can gain.

Sample Input

1
1

10
3 1 4 1 2 5 9 6 3 5

4
1000000000 250 505 999999999

5
2 4 6 8 10

Sample Output

0

8

3

1
